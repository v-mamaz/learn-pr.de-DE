## <a name="what-is-encryption"></a><span data-ttu-id="d11ff-108">Was ist Verschlüsselung?</span><span class="sxs-lookup"><span data-stu-id="d11ff-108">What is encryption?</span></span>

<span data-ttu-id="d11ff-109">Bei Verschlüsselung geht es um die Konvertierung bedeutungshaltiger Informationen in etwas, das bedeutungslos erscheint, wie eine zufällige Abfolge von Buchstaben und Ziffern.</span><span class="sxs-lookup"><span data-stu-id="d11ff-109">Encryption is about converting meaningful information into something that appears meaningless, such as a random sequence of letters and numbers.</span></span> <span data-ttu-id="d11ff-110">Bei einer Verschlüsselung wird ein bestimmter **Schlüssel** als Teil des Algorithmus verwendet, der die verschlüsselten Daten erstellt.</span><span class="sxs-lookup"><span data-stu-id="d11ff-110">The process of encryption uses some form of **key** as part of the algorithm that creates the encrypted data.</span></span> <span data-ttu-id="d11ff-111">Für die Entschlüsselung wird ebenfalls ein Schlüssel benötigt.</span><span class="sxs-lookup"><span data-stu-id="d11ff-111">A key is also needed to perform the decryption.</span></span> <span data-ttu-id="d11ff-112">Bei **_symmetrischen_** Schlüsseln wird für die Verschlüsselung und Entschlüsselung der gleiche Schlüssel verwendet, während bei **_asymmetrischen_** Schlüsseln verschiedene Schlüssel verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="d11ff-112">Keys may be **_symmetric_**, where the same key is used for encryption and decryption, or **_asymmetric_**, where different keys are used.</span></span> <span data-ttu-id="d11ff-113">Ein Beispiel für die letztgenannte Variante ist ein Schlüsselpaar aus einem **öffentlichen und einem privaten** Schlüssel, das in digitalen Zertifikaten verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="d11ff-113">An example of the latter is the **public-private** key pairs used in digital certificates.</span></span>
