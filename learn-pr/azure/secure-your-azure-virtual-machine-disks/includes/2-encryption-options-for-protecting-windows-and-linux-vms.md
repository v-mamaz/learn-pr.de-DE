## <a name="what-is-encryption"></a><span data-ttu-id="a1f96-108">Was ist Verschlüsselung?</span><span class="sxs-lookup"><span data-stu-id="a1f96-108">What is encryption?</span></span>

<span data-ttu-id="a1f96-109">Bei Verschlüsselung geht es um die Konvertierung bedeutungshaltiger Informationen in etwas, das bedeutungslos erscheint, wie eine zufällige Abfolge von Buchstaben und Ziffern.</span><span class="sxs-lookup"><span data-stu-id="a1f96-109">Encryption is about converting meaningful information into something that appears meaningless, such as a random sequence of letters and numbers.</span></span> <span data-ttu-id="a1f96-110">Der Vorgang der Verschlüsselung verwendet eine bestimmte Form von **Schlüssel** als Teil des Algorithmus, der die verschlüsselten Daten erstellt, und zum Durchführen der Entschlüsselung wird ebenfalls ein Schlüssel benötigt.</span><span class="sxs-lookup"><span data-stu-id="a1f96-110">The process of encryption uses some form of **key** as part of the algorithm that creates the encrypted data, and a key is also needed to perform the decryption.</span></span> <span data-ttu-id="a1f96-111">Schlüssel können **_symmetrisch_** sein, dann wird für die Verschlüsselung und die Entschlüsselung der gleiche Schlüssel verwendet, oder **_asymmetrisch_**, dann werden verschiedene Schlüssel verwendet, wie etwa bei den **öffentlich-privaten** Schlüsselpaaren, die in digitalen Zertifikaten verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="a1f96-111">Keys may be **_symmetric_**, where the same key is used for encryption and decryption, or **_asymmetric_**, where different keys are used such as in the **public-private** key pairs used in digital certificates.</span></span>
