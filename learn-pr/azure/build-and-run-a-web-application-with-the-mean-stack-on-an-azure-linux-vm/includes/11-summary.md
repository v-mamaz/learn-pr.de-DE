In diesem Modul haben Sie eine neue Ubuntu Linux-VM in Azure erstellt. Unter Verwendung dieser neuen Linux-VM haben Sie den MEAN-Stapel für die Webentwicklung installiert: MongoDB für einen Datenspeicher, Express für Webanforderungsrouting, AngularJS für clientseitige Webanwendungsfunktionalität und Node.js für serverseitige Webanwendungsfunktionalität. Mit den vorhandenen MEAN-Stack-Komponenten verfügen Sie über einen vollständigen und umfassend dokumentierten Webstapel, der für das Erstellen Ihrer nächsten Website verfügbar ist.
